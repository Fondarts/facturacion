# Archivos de Entrenamiento de Tesseract

Para que Tesseract OCR funcione correctamente, necesitas descargar los archivos de entrenamiento de idioma.

## Instrucciones:

1. Descarga el archivo `spa.traineddata` (español) desde:
   https://github.com/tesseract-ocr/tessdata/raw/main/spa.traineddata

2. Coloca el archivo en esta carpeta: `app/src/main/assets/tessdata/spa.traineddata`

3. (Opcional) Si quieres soporte para inglés también, descarga `eng.traineddata`:
   https://github.com/tesseract-ocr/tessdata/raw/main/eng.traineddata

## Nota:
La aplicación copiará automáticamente estos archivos al directorio de la app en el primer uso.
